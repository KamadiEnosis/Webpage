# My First Web Page

This is a simple, semantic HTML web page created as a basic demonstration of HTML elements and structure.

## Features

- Uses semantic HTML tags like `<header>`, `<nav>`, `<main>`, `<article>`, and `<section>`.
- Includes a navigation bar with links (Home, About, Contact us).
- Contains both unordered and ordered lists to present information.
- Demonstrates linking to an external website (Facebook).
- Simple headings and paragraphs to organize content.

## File

- `index.html` - The main HTML file containing the web page code.

## Usage

To view the web page:

1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
